    class Dirt::Role

    Roles are part of the DCI system design paradigm. They decorate a model object to extend its behaviour for the context of a single interaction.

def method_missing(method, *args, &block)
  if @decorated.respond_to?(method)
    @decorated.send(method, *args, &block)
  else
    super
  end
end
